# Chapter 07 Message Hierarchy

## Audience type

Academic robotics.

## Presenter purpose

Reframe the tutorial historically and explain why task essence is formalized as a state transition.

## Central thesis

LfO moved beyond motion copying by treating the essence of a demonstrated task as a transition between task-relevant states.

## Supporting claims

1. Diachronic discussion shows the shift from trajectory imitation to essence extraction.
2. Top-down LfO starts from task purpose rather than low-level motion.
3. Object and task recognition identify the entities and relations needed for state.
4. `T: s_t -> s_{t+1}` is the core abstraction that preserves intent.

## Final takeaway

The essence of a task is not the path taken; it is the state change that the path accomplished.
