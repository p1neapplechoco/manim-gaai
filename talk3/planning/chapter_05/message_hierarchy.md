# Chapter 05 Message Hierarchy

## Audience type

Academic robotics.

## Presenter purpose

Explain how grasping becomes a reusable skill-agent family by representing grasp purpose and closure rather than memorizing object-specific grasps.

## Central thesis

Grasp agents generalize across object shape when the task representation uses grasp purpose, closure, contact webs, and learned shape-to-action mappings.

## Supporting claims

1. Grasp taxonomies motivate but do not fully define executable grasp skills.
2. Closure theory converts grasp purpose into a condition the agent can pursue.
3. Contact-web agents and superquadric shape models organize object variation.
4. CNN and RL training support transfer across objects of the same grasp purpose.

## Final takeaway

The same grasp agent can handle many shapes because it targets a stable grasp condition, not a single hand pose.
