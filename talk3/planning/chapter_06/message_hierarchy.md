# Chapter 06 Message Hierarchy

## Audience type

Academic robotics.

## Presenter purpose

Show how LfO components combine into an errand robot system rather than remaining isolated modules.

## Central thesis

An errand robot needs recognition, task models, grasp/manipulation agents, and humanoid control to operate as one integrated LfO pipeline.

## Supporting claims

1. The errand project is the system-level testbed for modular LfO.
2. Humanoid execution exposes why body-specific control must stay local.
3. Key components form a pipeline from perception to task model to skill agents.
4. Integration reveals the need for a deeper definition of task essence.

## Final takeaway

The errand robot system demonstrates the architecture, then forces the question of what exactly the task model preserves.
