# Chapter 08 Message Hierarchy

## Audience type

Academic robotics.

## Presenter purpose

Show that the state-transition principle generalizes across domains, but the state representation itself is domain specific.

## Central thesis

LfO is domain-general at the level of state transitions, but each domain must choose state variables that preserve the relevant task differences.

## Supporting claims

1. Polyhedral manipulation uses face-contact state.
2. Machine assembly uses part-mating state.
3. Knot tying uses topological state.
4. Dance uses key pose and foot-contact state.
5. Synchronic comparison reveals a shared transition principle across different coordinate systems.

## Final takeaway

The invariant is not a universal state vector; the invariant is the idea of task-relevant state change.
