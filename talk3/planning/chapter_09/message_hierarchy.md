# Chapter 09 Message Hierarchy

## Audience type

Academic robotics.

## Presenter purpose

Synthesize the full tutorial into the final factorization: representation/encoding on one side, adaptive skill execution on the other.

## Central thesis

LfO transfer succeeds when task intent is represented as state transitions, grounded correctly from observation, and realized by available feedback-driven skill agents.

## Supporting claims

1. Imitation learning fails when trajectory equality is mistaken for task equivalence.
2. Symbolic teleoperation separates GPT/VLM task encoding from skill-agent decoding.
3. Cerebrum/cerebellum analogy explains why high-level intent and low-level reaction are separated.
4. Vision-only observation is defensible because force feedback is learned locally, not copied.
5. Transfer is bounded by representation, grounding, and skill-library coverage.

## Final takeaway

Generalist robotic behavior comes from factorizing invariant task state change from adaptive robot-specific execution.
